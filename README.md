# Coffee Prediction Analysis Model
This is a python group project whereby we shall be taking a coffee export dataset, cleaning the dataset, working on it then creating a model using various libraries

1. Problem Definition & Objective (10 marks)
Guidelines:
• Clearly define the problem statement.
• State the goal of your project.
• Justify the relevance and potential impact of solving this
problem.
• Specify whether it's a classification, regression, clustering, or
other AI task.
Deliverable: One paragraph problem description and project
objective.
2. Data Acquisition (10 marks)
Guidelines:
• Source relevant, clean, and useful data.
o Sources: Kaggle, UCI, government portals, APIs, web
scraping, etc.
• Describe the dataset: size, format, number of features, time
span, etc.
• Include data collection method if not using a public dataset.
Deliverable: Brief summary of data source, description, and
justification.

Guidelines:
• Handle missing values, duplicates, and outliers.
• Convert data types where necessary.
• Normalize, encode, or transform variables.
• Split the dataset into training and test sets.
Deliverable: Code + explanation of key cleaning steps.

Guidelines:

5. Feature Engineering (10 marks)
Guidelines:

• Scale numerical features (if needed).

Guidelines:
